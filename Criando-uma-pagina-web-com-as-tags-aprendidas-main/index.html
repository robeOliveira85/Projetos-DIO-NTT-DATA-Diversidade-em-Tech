O que consiste o HTML?

O HTML, basicamente, consiste em três partes. Temos a tag geral, sendo a identificação para o html, que estará envolvendo a tag head e body.
Na tag head vai constar tudo aquilo que o usuário não terá contato visual, diferente da tag body, na qual vai constar todo o conteúdo que o usuário vai visualizar em um site.

Falando sobre as tags

As tags são elementos usados para informar qual o tipo de estrutura que estará sendo criada, por exemplo, se eu quiser inserir um título em minha aplicação, posso utilizar, semanticamente falando, a tag 'h1'.
As tags funcionam de maneira que todo o conteúdo inserido dentro de uma tag vai adiquirir a sua respectiva funcionalidade, por exemplo, um texto dentro da tag 'b' vai tornar esse texto em negrito.
A maioria das tags utilizadas precisam ter uma abertura e fechamento, para indicar o seu ínicio e fim, entretando, existem algumas tags que abrem e fecham na mesma tag, por exemplo, o 'input'

Atributos das tags

Os atributos são propriedades que uma tag pode ter, indo desde os atributos mais específicos aos mais gerais.
Por exemplo, na tag 'h1' posso ter um atributo chamado id que vai ser o identificador desse elemento, podendo ser identificado pelo javascript para manipular as interações desse elemento. Um exemplo de atributo geral, seria a id, pois ela vai ser usada para identificar qualquer tag e um exemplo de tag específica seria no caso de um input, vai ter um atributo específico chamado 'type' para identificar o tipo do input.

Textos

Voltados para as tipografias, temos as tags h1 até h5, sendo organizadas por título e subtítulos, sendo a primeira tag (h1) o nível maior, considerado o título da página.
Temos a tag 'p', considerada a tag do paragráfo.
Temos a tag de citação, sendo a tag blackquote.
